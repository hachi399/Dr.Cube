キューブ博士へようこそ！このサイトでは、我々群馬高専チームがDcon2026優勝のために作るプロダクトについて紹介しています。

2026年1月13日に第一プロトタイプ完成予定で、現在2025/11/13時点ではソフトウェアの開発途中です。

チーム構成は4人（ソフト2名 / マーケ1名 / マネージャー1名）。 プログラミング経験はすべて独学ですが、気合と情熱だけで最後まで走り切ります！

作品内容

今回のプロダクトでは、学習、観光、福祉など幅広い分野で活用できるAIシステムを開発しています。

進捗状況

以下URLが現在稼働中の AI解析サイトです。ここからさらに何百もの機能を追加し、最終的にはキューブ型デバイスで動作させます。「なぞって解析！」の部分はまだスマホ未対応なのでパソコン上で試していただけたらと思います。

AI解析サイトを見る

メッセージ欄に知りたい内容を打ち込むとAIが回答します。

三本線マークをクリックするとAI解析サイドバーがにゅるっと表示されます。

カメラONでインカメラが起動し、自動ピント → 自動撮影 → AI解析が行われます。 撮影時は周囲の明るさを十分にしてください。レンズが汚れていると正しく解析できない場合があります。

解析テキストをドラッグすると、その内容の追加解析がサイドバーに表示されます。

GitHub と Render を連携し、Raspberry Pi から常時アクセスすることで Render のサーバー休止を防ぎ、起動待ち時間ゼロを実現しています。

GitHub リポジトリはこちら：

https://github.com/hachi399/Dr.Cube
